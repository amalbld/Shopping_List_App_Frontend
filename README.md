# Shopping List App 🛒

Application frontend React de gestion de liste de courses — ajout, suppression et suivi des articles.

---

## 🛠 Stack technique

| | Technologies |
|--|-------------|
| **Frontend** | React · JavaScript · CSS · HTML |
| **State** | React State (useState) |
| **Outils** | Create React App · npm · Git |

---

## ✨ Fonctionnalités

- ➕ Ajouter un article à la liste
- 🗑️ Supprimer un article
- ✅ Marquer un article comme acheté
- 📋 Affichage dynamique de la liste en temps réel

## 🚀 Lancer le projet

### Prérequis

- Node.js 18+ & npm

### Installation & démarrage

```bash
git clone https://github.com/amalbld/Shopping_List_App_Frontend
cd Shopping_List_App_Frontend
npm install
npm start
```

L'application démarre sur **http://localhost:3000**
